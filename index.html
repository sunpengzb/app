日报

  今天学习了 git的很多东西，比如：时光机穿梭里面的：版本回退 先用git log 命令可以看到从最近到最远的提交日志，然后再输入 git reset –hard HEAD^ 命令 来回退到上一个版本 ， 再输入git log 以后，就会看到命令行里面少了一个提交过的日志 ， 如果还想回到一开始的状态，就输入 git reset –hard “commit id”，来进行版本返回。重返未来，用git reflog查看命令历史，以便确定要回到未来的哪个版本。
  接下来就是工作区和暂停区了 用 git add . 可以让全部的文件添加进暂停区 ， 然后再用git commit 提交更改，把暂停区的所有内容提交到当前分支。 再然后就是添加远程库 ， 在github里面添加仓库让文件可以进入github里面。
